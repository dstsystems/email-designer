# email-designer
A technical test aimed at potential Web Designer candidates. A chance for them to show off their Email development skills.

The idea behind this technical test is for you to show us what you can do. You are free to use any framework and libraries you wish but the finished product must look like the JPG found with this test.

# Instructions

1. Download the ZIP file from the DST Systems GitHub account: https://github.com/dstsystems/email-designer
2. The look of the final page can be views in the email-test.jpg file
3. All images must be from the JPG file.
4. Your final build must be responsive and be viewable in mobile devices
5. When you've finished, zip up all your files and name the zip file your name

### Additional instructions
1. All fonts must be email friendly
2. All colour codes can be pulled from JPG files

# What we're looking for

We would you like you to demonstrate your understanding of email builds and the problems that arise as a result. It's up to you to make sure the email renders in all email clients.
